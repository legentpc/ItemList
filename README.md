# SkyBlock Item List

An item list, similar to [REI](https://modrinth.com/mod/rei/) and [JEI](https://modrinth.com/mod/jei/), made
specifically for Hypixel SkyBlock.

Check out [the gallery](https://modrinth.com/mod/skyblock-item-list/gallery) to see the mod in use!

# Features

* Main Item Panel
	* Contains all items, attributes, enchantments, mobs, NPCs, pets, potions, and runes!
	* Can be searched, resized, and filtered based on item type.
* Recipe and Item Usage Lookup
	* Crafting Recipes,
	* Forge Recipes,
	* Kat Pet Upgrade Recipes,
	* NPC Shop Recipes.
* Favorites Panel
	* Add your favorite SkyBlock items.
	* Add frequently used recipes.
	* Pin the recipe you're currently working on.
* Built-in Search Bar Calculator
	* Bazaar (bz), Lowest BIN (lb), and NPC Sell (npc) price functions!
* Mod Compatibility API
	* Register exclusion zones to prevent items from being placed on top of widgets.
	* Hide the item list on custom screens.
	* Easily receive keybind events without needing to create and maintain problematic mixins.
	* Add custom buttons to recipes.
	* For more information, see the 
      [Plugin API](https://github.com/OperationPotato/ItemList/blob/master/src/main/java/com/operationpotato/itemlist/api/Plugin.java)
      on GitHub.

# Usage Guide
* See the "SkyBlock Item List" category in the Minecraft Controls screen to adjust keybinds.
* To toggle the overlay, while in the inventory, press `Ctrl + O` or `Cmd + O`.
* To change the scale of items, hold `Ctrl` or `Cmd` as you scroll over the item list.
* Hovered Item Interactions
	* When hovering over an item, you can view its recipes and usages or favorite it.
	* To view its recipe, press `R`.
	* To view its usages, press `U`.
	* To favorite the item, press `F`.
* Access the Config through [Mod Menu](https://modrinth.com/mod/modmenu) or by clicking the "⚙" button next to the search bar.
